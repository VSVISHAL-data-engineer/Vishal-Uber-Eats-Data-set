# 🍽️ Uber Eats Bangalore Restaurant Intelligence

## Project Overview
Bangalore Uber Eats restaurant data-வ analyse பண்ணி,
business decisions எடுக்க help பண்ற Decision Support System.

## Tech Stack
- Python
- Pandas
- SQLite
- Streamlit

## Project Steps
1. Data Cleaning (23,158 restaurants)
2. Database Setup (SQLite)
3. SQL Queries (10 Business Questions)
4. Streamlit Dashboard

## How to Run
```bash
python data_cleaning.py
python database_setup.py
python sql_queries.py
streamlit run app.py
```

## Business Questions Answered
1. Top locations by rating
2. Most saturated areas
3. Online order vs rating
4. Table booking impact
5. Price range vs satisfaction
6. Price segment performance
7. Popular cuisines
8. Highest rated cuisines
9. Niche cuisine opportunities
10. Cost vs rating correlation
